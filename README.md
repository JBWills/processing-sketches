My personal Kotlin Processing sketches workbook

The output of these sketches is into SVG, so I can plot them using the [AxiDraw plotter](https://www.axidraw.com/).

## Getting Started

### Downloading non-gradle libraries
There are a few libraries you can't get via gradle, so download the .jar files manually and put them in the /third-party dir.
1. svg
    - [batik-all-1.14.jar](https://archive.apache.org/dist/xmlgraphics/batik/binaries/)
        - install this into the processing-svg subdirectory under /third-party
2. controlP5
    -  [ControlP5](https://github.com/sojamo/controlp5)
        - place the controlP5.jar file directly in the third-party dir.
